CNGM and CREMA work together to satisfy our gaming needs.

Welcome to the System

Here’s how to get up and running in a few minutes.

None of this is mandatory right now — everything can be set up later from the Connect and Tools menus.

How the ecosystem works

Steam games → imported here via API, launched by the Steam Client.

GOG & Epic games → installed, configured and launched by GRINDER.

You can also add any other game or app manually — no problem!

Step 1 — Import Your Steam Library

Steam games launch via the Steam Client. Use your API key to import your full library so CNGM can show artwork and metadata.

1. Free API Key: steamcommunity.com/dev/apikey
2. SteamID64: your profile URL or steamid.io
★ Game Details must be set to Public

Step 2 — GOG & Epic → Install GRINDER

GRINDER is your GOG and Epic installer and launcher. It handles downloading, Proton configuration, and launching — CNGM just needs to know where it is.

Place GRINDER.AppImage in the same folder as CNGM. That’s it — they connect automatically.

Step 3 — Fetch Artwork & Metadata

Pulls covers, hero art, logos, HLTB, ProtonDB and descriptions for all your games in one pass. Run this after importing your Steam library.

Step 4 — API Keys (Optional)

SteamGridDB — custom fan-made logos and hero art.

Free key at steamgriddb.com/profile/preferences/api

IGDB — richer metadata and scraping fallback for non-Steam games.

Free credentials at dev.twitch.tv — register a Twitch app.

Step 5 — CREMA Companion

Place CREMA.AppImage in the same folder as CNGM. A Go Fullscreen With CREMA button will appear at the bottom of the sidebar — your fullscreen, gamepad-driven companion sharing the same library.

Step 6 — System Menu Shortcuts

Register CNGM, GRINDER and CREMA in your desktop launcher (GNOME, KDE…) with proper icons. Re-run if you ever move the folder.

Don’t worry! You don’t need to do any of this right now. All options are always available in Connect and Tools whenever you’re ready.

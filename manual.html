Introduction

Crema — the reddish-brown, frothy foam that sits atop a freshly brewed espresso — is created by water, coffee oils, and pressurized CO₂ during high-pressure extraction. It signals freshness, quality, and craft. That's the metaphor this project lives by.

Cafe Neurotico Game Manager (CNGM) is the espresso machine: exacting, thorough, neurotic about your library. It imports from stores, scrapes artwork, manages metadata, and keeps everything organized with obsessive precision.

CREMA is the sweet foam on top: the fullscreen experience you drop into when it's time to play. Grab a controller, pick a game, and enjoy — CREMA handles the rest.

How they connect

Both apps share the same database. CNGM builds and enriches it; CREMA reads and presents it. Set up your library in CNGM first, then launch CREMA for the couch experience.

Getting Started

On first launch, CNGM opens to an empty library. Follow these steps to get up and running:

  1. 1
    Launch the app. The main window opens with the Library panel active and an empty game list.
  2. 2
    Set up your API keys. Open the Import panel to configure Steam, IGDB, and SteamGridDB credentials. See each import method below for where to get keys.
  3. 3
    Import your games. Use any of the store import methods or add games manually.
  4. 4
    Run Batch Scrape. Go to Tools → Batch Scrape Missing Data to automatically download covers, hero art, and metadata for your entire library.
  5. 5
    Launch CREMA. Click the Launch CREMA button to open the fullscreen launcher — your library is already there, ready to play.
API Key Sources

Steam API Key → store.steampowered.com/dev/apikey (requires a public profile)
IGDB → dev.twitch.tv — create an app, get Client ID + Client Secret
SteamGridDB → steamgriddb.com → Account → Preferences → API Key

Importing Games

CNGM supports six import paths. All methods add games to the same shared library.

Steam

  • Requires your SteamID64 and a Steam API Key
  • Your Steam profile must be set to Public
  • Imports the full owned game list with Steam App IDs pre-filled
  • Launch commands are generated automatically for Steam games

GOG

  • Uses a secure OAuth login window — your credentials are never stored
  • Fetches the full GOG library for the logged-in account

Epic Games

  • Direct import from your Epic Games library

Heroic Games Launcher

  • Syncs games already imported into Heroic
  • Heroic must be installed and configured with your accounts

CSV Import

  • Download the CSV Template from the Import panel
  • Fill in game details using any spreadsheet app
  • Import the completed file to batch-add custom games
  • Use Export CSV to back up or migrate your library data

Manual Add

  • Click Add Game to open the game creation form
  • Fill in the title and any other known fields
  • Use the Scraping Engine afterwards to fill in artwork and metadata

Search

The search bar filters across game titles, descriptions, genres, developers, publishers, and series — results sorted with title matches first.

Gaming History

CNGM tracks which games you've launched and when. The Recently Played section shows your most recent games at the top of the library. Configure how many entries appear (5, 9, or 18 games) or disable it entirely. Clear history at any time from the Tools panel.

Metadata & Game Details

Click any game to open its full detail page. The editor panel lets you view and modify every field.

Auto-Fetch (IGDB)

  1. 1

  3. 3
    If multiple matches are found, a selection list appears. Choose the correct entry.
  4. 4
    Metadata is applied. Review and save any adjustments.

Batch Scraping

The Batch Scrape tool scans your entire library and downloads missing artwork and metadata automatically. Progress is shown per-game. This is the fastest way to populate a freshly imported library.

SteamGridDB Integration

For games not found by the default scraper, or when you want alternative artwork, use SteamGridDB. Enter your API key once; then for any game, select Custom Cover (SteamGridDB) to browse community-made assets and apply them with a single click.

Trailer Downloading

  1. 1
    Open a game's detail page and click Download Trailer.
  2. 2
    CNGM searches YouTube for the game's trailer.
  3. 3
    Select the best result from the list.
  4. 4
    The trailer downloads locally. CREMA will play it from disk — no internet needed at launch time.

Manual Asset Assignment

Click the folder icon next to any asset slot to browse your disk and assign a custom image. Useful for physical games, emulation, or any title not indexed by online databases.

Tools & Settings

Appearance

  • Theme — 50+ color schemes across six categories. See the Theme Catalog.
  • UI Scale — 0.8×, 0.9×, 1.0×, 1.1×, or 1.2× zoom. Useful for high-DPI displays.
  • Language — English or Português (BR). Reloads the interface instantly.

Library Utilities

  • Update Library — Refreshes the game list from the database.
  • Batch Scrape Missing Data — Downloads artwork and metadata for all games that lack it.
  • Backup ZIP — Exports your entire library (database + images) as a ZIP archive.
  • Clean Images — Removes image files no longer referenced by any game.
  • Clear All Images — Wipes all visual assets (cover, hero, screenshots) from the library.
  • Clear Gaming History — Resets the recently-played list.
  • Wipe Secure Browser Data — Clears stored login sessions (e.g., GOG OAuth).

First-Time Setup

The first time you launch CREMA, a two-step setup wizard guides you through the essentials. This screen only appears once; all choices can be changed later via the System Menu.

  1. 1
    Choose your Start Screen style. Three visual modes are shown as preview cards: List (vertical category list), Carousel (horizontal scrolling), or Grid (3×3 category tiles). Navigate with the D-pad or arrow keys.
  2. 2
    Choose your input preference. Select Gamepad or Keyboard. CREMA will auto-detect input type during normal use, so this is just a hint for the default hints.
  3. 3
    Confirm. Press A or Enter to finish. Your library appears immediately.

Navigation Modes

CREMA offers three distinct ways to browse your library. Switch between them anytime via System Menu → Start Screen.

List Mode
A vertical category menu with a large hero media section. Clean and fast — ideal for large libraries or keyboard users. D-pad up/down navigates; left/right changes category.
Carousel Mode
Categories scroll horizontally as large cards with hero art. A cinematic browsing experience best enjoyed on a TV.
Grid Mode
Categories displayed as a 3×3 tile grid with a hero banner above. Touch-screen friendly and visually dense.
Browse Mode

Within each category, toggle between List View (scrollable game list with metadata sidebar) and Gallery View (cover art grid with hero banner) via the System Menu → Browse Mode.

Global Search

Press Y (gamepad) or Y (keyboard) to open the on-screen keyboard. Type to filter games by title, description, genre, developer, publisher, or series. Press A or Enter to confirm; the list filters instantly. Press B or Esc to clear and return.

Gallery View Navigation

  • D-pad / Arrow keys — move between game tiles
  • LB / RB or , / . — switch category left / right
  • The selected tile is always kept centered on screen with smooth scrolling
  • The top row shows Recently Played games
  • Store badges (Steam, GOG, etc.) appear at the bottom-right of each tile

Gamepage

Press A on any game (in List View or Gallery View) to open its full Gamepage — an immersive detail view with all available art and metadata.

Layout

  • Hero — full-width animated banner (Ken Burns pan-zoom)
  • Action Buttons — Play, Favorite, Want to Play, accessible directly from the hero area
  • Cover Art — displayed on the left of the content area
  • Descriptions — short summary followed by the full description
  • Game Stats — Developer, Publisher, Genre, HLTB, ProtonDB, Metacritic, Co-op, Players, Series, Similar Games
  • Screenshots Banner — scrollable screenshot slideshow; press X to open fullscreen

Trailer Playback

If a trailer has been downloaded via CNGM, press X in the hero area or select Watch Trailer to open the trailer popup. The BGM fades out automatically during playback and fades back in when you close the overlay. Press B to return.

Games Without Artwork

If a game has no scraped artwork or metadata, CREMA shows a styled placeholder with a hint: press SELECT → Scraping Engine to download assets, or use CNGM for more detailed editing tools.

Scraping Engine

The scraper fetches data from the Steam database and IGDB. Available actions:

  • Scrape All (Default) — cover, screenshots, and full metadata in one pass
  • Cover Only — download cover art without changing other data
  • Screenshots Only — download screenshot set only
  • Metadata Only — fetch text data (developer, genre, descriptions, etc.)
  • Custom Cover (SteamGridDB) — browse and apply community cover art
  • Remove Cover / Screenshots / Metadata — delete specific asset types
  • Clear All Scraped Data — wipe everything for a clean slate
Refine Search

If the automatic scrape doesn't find the right game, select Refine Search to type a custom search term and pick from a list of matches manually.

Jukebox

CREMA includes a full music player. Open it from the System Menu or press START → Jukebox Mode.

Adding Music

Place audio files (MP3, FLAC, OGG, WAV) in the CREMA_CUSTOM_MUSIC folder inside the CREMA directory. Close and reopen the Jukebox to scan for new tracks.

Library Organization

  • All Songs — complete track listing
  • Artists — grouped by artist tag
  • Albums — grouped by album tag
  • Playlists — your custom playlists
  • Shuffle All — randomize and play the entire collection

Playlist Management

  • Create a new playlist with + Create New Playlist
  • Add any song to a playlist via its song options menu
  • Rename, duplicate, or delete playlists from the playlist options menu

Now Playing

The Now Playing widget shows the current track's cover art, title, and artist with animated visualizer bars. The Up Next section shows what's coming.

Screensaver

CREMA activates a screensaver after a configurable idle period. Configure via System Menu → Screensaver.

Settings

  • Delay — 1 to 30 minutes of inactivity before activation
  • Mode — CN Wallpapers (curated art) or Screenshots (your scraped game screenshots)
  • View Now — manually trigger the screensaver immediately

Screensaver Display

  • Full-screen image slideshow with Ken Burns animation
  • Floating clock (12-hour format)
  • Game title and store badge overlays
  • Quick-action hints: A Launch, Y Favorite, X Want to Play

Waking Up

Press any button on the gamepad or keyboard to exit the screensaver and return to your library. Configure the wake input method via System Menu → Wake Method.

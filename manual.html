1. Manifesto & Philosophy

Your Library. Manageable. Searchable. Portable.

I wanted an app that could organize and launch all my games — from Steam, Heroic (Epic and GOG), Flatpak, Emulators, or whatever — in just one place.

I wanted it to be complete with all the features I’ve always wished I had in a Linux game manager: checking YouTube videos, HowLongToBeat, and ProtonDB right from inside the app, among other things.

I wanted it to be simple, beautiful and themeable.

And most importantly, I wanted it to be completely PORTABLE and BACKUPABLE.

The Local-Only Philosophy

CNGM is built with a strict Offline-First, Local-Only philosophy for your media. Once you fetch artwork, covers, logos, or download a trailer, it is saved securely into your local GameManagerConfig folder.

This means your library will never suffer from “link rot” or broken web images if a storefront changes its API. It also means you can pack up the entire folder, put it on a thumb drive, and take your flawlessly curated library to another PC entirely.

The Launch Command

The crucial feature. Inside the Edit Details screen, there is a field where you can put a command to launch any game. Anything goes — if you can run it on Linux, you can run it from here.

If you imported your games via Steam or Heroic Sync, the launch commands are filled in automatically, even if the game is not installed. Clicking PLAY will prompt you to install if needed.

2. Quick Start

CNGM and CREMA are two apps that share one library database and live in the same folder. CNGM is the desktop manager; CREMA is the fullscreen couch companion. Neither requires system installation — just run the AppImage.

The Welcome & Setup Guide

Every time CNGM launches, it shows a Welcome & Setup Guide screen that walks you through the recommended first-run workflow step by step. Each step has live action buttons built right into the panel — you can sync Heroic, enter your Steam credentials, run a batch fetch, save API keys, and register your system shortcuts all without leaving the screen.

What the Welcome Screen Contains

  • Step 1 — Connect Your Stores: Sync Heroic Games (Epic, GOG, Amazon) with one button. Paste your SteamID64 and API Key inline to fetch your Steam library directly from the screen.
  • Step 2 — Fetch Artwork & Metadata: Runs Batch Fetch with an inline progress bar — no need to open the Tools menu.
  • Step 3 — API Keys (Optional): Paste and save your SteamGridDB key and IGDB credentials directly, with a live connection test for IGDB.
  • Step 4 — CREMA Companion: Explains where to place CREMA.AppImage to activate the fullscreen companion.
  • Step 5 — System Menu Shortcuts: Registers both CNGM and CREMA in your desktop launcher with a single button press.

The action buttons in the Welcome Screen perform their tasks without closing the screen, so you can complete the entire setup in sequence from top to bottom. Only Let’s Go and Open User Manual dismiss it.

Don’t show again: A checkbox at the bottom of the Welcome Screen lets you opt out of seeing it on future launches. If you ever want to bring it back, go to Tools → System → Show Welcome & Setup Guide. This also resets the “don’t show again” flag so the screen appears on the next launch as well.

Your Files

On first launch, CNGM creates a GameManagerConfig folder next to the app. Inside:

  • game_library.db — your SQLite library database (all your games and metadata)
  • images/ — covers, hero art, logos, icons, screenshots
  • trailers/ — locally downloaded MP4 trailers

Portability tip: Back up the entire GameManagerConfig folder and you have your complete library — artwork, trailers, metadata, everything. Restore it anywhere, run the app, and it all works immediately.

Recommended First-Run Workflow

  1. On first launch, the Welcome & Setup Guide appears automatically — follow the steps there for the fastest setup.
  2. Open Connect and sync your stores — Heroic first, then Steam.
  3. Open Tools → Library and run Batch Fetch Missing Data to pull artwork for all synced games at once.
  4. Set up SteamGridDB (free API key at steamgriddb.com) and IGDB (free Twitch dev key at dev.twitch.tv) in the Connect menu for richer metadata and custom logo art.
  5. Place CREMA.AppImage in the same folder as CNGM — the Go Fullscreen With CREMA button appears automatically at the bottom of the sidebar.
  6. Open Tools → System and click Add to Application Menu to register both CNGM and CREMA in your desktop launcher with proper icons — no terminal required.

3. CNGM — Adding Games & Store Sync

Click Connect in the sidebar to reach the store sync menu. All data is stored locally — no cloud accounts are required to use the app.

Heroic Sync (Epic / GOG / Amazon) — Recommended

If you use Heroic Games Launcher, this is the best way to import your library. CNGM reads Heroic’s local configuration files directly — no login window, no API key. It imports everything you own, installed or not, with the correct native launch commands already filled in.

  • Sync Heroic Games: Imports all Epic, GOG, and Amazon libraries from your local Heroic installation in one shot.
  • Refresh Heroic New Games (in Tools → Library): Opens Heroic, waits for you to update its library, and auto-syncs when Heroic finishes — use this after buying a new game in Epic or GOG via Heroic.

Amazon Games via Heroic are fully supported. If you have Amazon Games connected in Heroic, they appear in CNGM automatically under the Amazon category, with launch commands ready.

Steam API Import

Valve’s scraping protection makes direct scraping unreliable. The official Steam API is the right way:

  1. Get a free API Key at steamcommunity.com/dev/apikey
  2. Find your 17-digit SteamID64 (from your profile URL or steamid.io)
  3. Enter both in the Connect menu and click Fetch Steam Library

Important: Your Steam profile’s Game Details must be set to Public for the API to return your game list.

IGDB Integration

IGDB (Internet Game Database) enriches metadata with franchise info, ratings, and additional game data. Get free credentials at dev.twitch.tv: register a Twitch developer app and copy the Client ID and Client Secret into the Connect menu. Once set, Auto-Fetch automatically pulls IGDB data alongside Steam data.

GOG Direct Login

Opens a secure internal browser login window. Log in and close it — CNGM will import your owned GOG titles. However, this method cannot auto-generate launch commands. You will need to fill them in manually for each game. The strongly recommended alternative is connecting GOG to Heroic and using Heroic Sync, which provides launch commands automatically.

Adding Games Manually

Hit + Add Game in the sidebar. Enter the game’s title — a new blank entry is created in the database. From there, open Edit Details and click ✨ Auto-Fetch to pull all metadata and artwork automatically. For Physical, Emulation, or other non-store games, type any terminal command into the Launch Command field.

Advanced: Custom Stores & Tags

The Store field is plain text with no restrictions — you can type anything you want there. This opens up a neat trick: create your own virtual stores for platforms that don’t have a dedicated sidebar button, like itch.io, GeForce Now, Xbox Cloud Gaming, Humble Bundle, GamePass, or anything else.

The search bar searches every field in the database, including Store. So a game with Store: itch.io will appear when you type itch.io in the search box, giving you instant filtered access to all your itch.io games.

Recommended pattern: Use Others, itch.io instead of just itch.io. The comma-separated value means the game still shows under the Others sidebar filter (which matches anything containing the word “others”), and is also searchable by typing itch.io. Using just the custom name alone works fine too — it will be searchable but won’t appear under any sidebar filter button.

Avoid accidental keyword matches: Don’t include steam, epic, gog, heroic, amazon, physical, others, emulation, or apps unintentionally in a custom name — the sidebar filters use a simple text match, so a store named steam-like would show up under the Steam filter.

CSV Import / Export

Prefer bulk-adding via spreadsheet? Download the CSV Template from Tools → Library, fill it in, and import it back. All standard fields are supported. Export CSV does the reverse — dumps your entire library to a spreadsheet for editing or archiving.

4. CNGM — Views & Navigation

Three main views, one persistent sidebar. The sidebar filter applies instantly from anywhere — even from deep inside a Gamepage, clicking a filter button snaps you back to the last grid view you were using without needing to navigate manually.

The List View

High-density, spreadsheet-style. Shows launch status, Favs, Want to Play, Store badge, Genre, and Release Year at a glance. Double-click any row to open its Gamepage. If Gaming History is enabled in Tools, your most recently played games are pinned to the top of the list.

The Gallery View

Visual cover-art grid. At the top, a dynamic Hero header cycles through hero art and screenshots from your currently filtered selection — a built-in animated screensaver effect using a Ken Burns zoom and crossfade. Your recent games appear first when history tracking is active. Each card shows the game cover, store badge, and an install status dot.

Sidebar Filters (See Section)

  • All Games — entire library
  • Installed — games currently detected as installed on this machine
  • Favs / Want — your personal curation lists
  • Steam / Epic / GOG / Amazon — filter by store
  • Physical / Others / Emulation / Apps — manual categories for non-store games

The CREMA Companion Button

If CREMA.AppImage (any capitalization) is in the same folder as CNGM, a Go Fullscreen With CREMA button appears at the bottom of the sidebar. Clicking it minimizes CNGM and launches CREMA as a separate fullscreen process. Both apps access the same game_library.db — any change in one is immediately reflected in the other.

5. CNGM — Installed Status

CNGM tracks whether each game is currently installed on your system and reflects it visually everywhere.

The Install Dot

A small colored dot overlays every cover card (top-left corner) and list row. Green = installed. Orange = not installed or status unknown.

How Detection Works

  • Steam games: Detected by scanning appmanifest_*.acf files across all your configured Steam library folders, including secondary drives. A live file watcher updates statuses automatically when you install or uninstall a game in Steam — no manual refresh needed.
  • Heroic games (Epic / GOG / Amazon): Read from Heroic’s installed.json files. Run Sync Heroic Games or Check Installed Games to refresh these.
  • Physical / Others / Emulation / Apps: A game in these categories is considered installed only if it has a Launch Command saved. Adding the command marks it installed immediately.

Install / Play Buttons

In Gallery and List view, uninstalled games show an INSTALL button. Clicking it:

  • Steam: Opens the Steam store install page for that game.
  • Heroic / Epic / GOG / Amazon: Opens the Heroic install page for that game.
  • Physical / Others / Emulation / Apps: Opens an Add Command dialog so you can type the launch command directly.

Check Installed Games in Tools → Library manually rescans your entire library and updates all statuses at once. Use it after a big Heroic session or if statuses look stale after syncing.

6. CNGM — The Gamepage

Double-click any entry to enter the immersive Gamepage — a premium storefront-style view for that game.

What You See

  • Hero Banner: A full-width cropped image at the top. The transparent Logo is overlaid at the center.
  • Action Bar: Play / Install button, Edit Details, Watch Trailer (if downloaded), FAV toggle, and WANT TO PLAY toggle.
  • Stats Panel: Release year, Developer, Publisher, Genre, HLTB, Metacritic, ProtonDB rating, Co-op info, Player count, Franchise / Series, and Similar Games.
  • Description: A plain-text summary first, followed by the full rich HTML description pulled from the Steam store page.
  • Screenshots: An animated Ken Burns slideshow at the bottom. Click any screenshot to enter a fullscreen slideshow. Click the fullscreen view to advance frames.

Live Toggles

The ★ FAV and ⚑ WANT TO PLAY buttons in the action bar toggle instantly and write directly to the database — no need to open Edit mode.

The Trailer Button

The 🎬 WATCH TRAILER button only appears if a trailer video has been downloaded locally. It never streams from the internet — if the button is there, the video plays instantly at full quality, entirely offline.

7. CNGM — Edit Details & Auto-Fetch

Click ⚙️ EDIT DETAILS from any Gamepage. Every field is editable and saved directly to the database on demand.

✨ Auto-Fetch

The single most powerful button. It runs a comprehensive scrape and pulls everything it can find in one go:

  • Cover art, Hero Art, Logo (with SGDB fallback if API key is set), Icon, up to 5 Screenshots
  • Short blurb description and full rich HTML store description
  • Release Year, Developer, Publisher, Genre, Franchise / Series
  • HLTB (HowLongToBeat main story hours)
  • ProtonDB compatibility tier (Platinum, Gold, Silver, Bronze, etc.)
  • Metacritic score, Co-op info, and Player count

Step-by-Step: Building the Ultimate Gamepage

  1. Auto-Fetch first — covers ~90% of everything automatically.
  2. If the Hero Art or Logo looks generic, click the SGDB button next to those fields to browse stunning custom fan art from SteamGridDB.
  3. Click ▶ WATCH TRAILER in the edit bar — search YouTube, pick the best result, and download it locally with one click.
  4. Save and return to the Gamepage. You now have a fully curated, completely offline, beautiful entry.

HLTB & ProtonDB Live Lookups

The 🔍 icons next to HLTB and ProtonDB fields trigger live API queries. If the lookup succeeds, the value drops into the field automatically. If it fails (unusual title, API momentarily down), CNGM opens a secure internal browser window directly to the search results page — read the value, close the window, type it in. No dead ends.

Asset Management (Left Column)

Each image asset (Cover, Hero, Logo, Icon) has its own row with four options: Auto (fetch from Steam CDN), SGDB (browse community art), Local (pick any file from your disk), and Delete (remove the current asset).

🧹 Clear Meta

Wipes all scraped metadata fields (description, stats, etc.) without touching images or launch commands. Useful when you want to re-scrape a game completely from scratch.

8. CNGM — Media, SGDB & Trailers

SteamGridDB (SGDB)

SteamGridDB is a massive community art archive with custom transparent logos and textless hero backgrounds. Get a free API key at steamgriddb.com and paste it into the Connect menu. Once configured, every SGDB button in the Edit screen opens a browsable grid of top-rated art for that asset type.

If the auto-search picks the wrong game (e.g., “Doom 2016” instead of “Doom 1993”), use the manual search box inside the SGDB window to hunt for the exact variation you want. Click an image to download and apply it immediately.

YouTube Trailers & The Local Rule

Clicking ▶ WATCH TRAILER in the edit bar queries YouTube for the game’s trailer. A list of results appears — pick the best one. CNGM uses yt-dlp internally to download the highest-quality MP4 to your local trailers/ folder.

The Local Rule: The 🎬 WATCH TRAILER button on the Gamepage only appears if the trailer has been downloaded to disk. CNGM never streams from the internet on the Gamepage. If the button is visible, the video plays instantly, offline, at maximum quality.

yt-dlp and ffmpeg are bundled inside CNGM — no extra installation required. Trailer downloads work out of the box.

9. CNGM — Tools & Settings

Click Tools in the sidebar. Cards are organized by function. Use the search bar at the top to jump directly to any setting by keyword.

🎨 Appearance

Theme Selection opens the full theme picker organized by category. Live-preview any color scheme instantly without committing. UI Scale zooms the entire interface from 50% to 150% — useful for Steam Deck or 4K displays.

🕒 Gaming History

Controls how many recently launched games CNGM pins to the top of the library views (OFF, 5, 10, 15, or 20 recent games). Clear History wipes the timestamps without touching any other data. Note: CNGM and CREMA maintain separate history counts, independently configured in each app.

✨ Library

  • Refresh Heroic New Games: Opens Heroic Games Launcher, watches for library changes, and auto-syncs when Heroic finishes updating. Use after buying a new game in Epic or GOG via Heroic.
  • Batch Fetch Missing Data: Scans all five asset types (Cover, Hero, Logo, Icon, Screenshots) for every game and silently pulls whatever is missing from Steam. Run it once after a big sync.
  • Check Installed Games: Manually rescans Steam appmanifests and Heroic installed files, updating the Installed status for your entire library in one shot.
  • Update Library: Forces a full library reload from the database, refreshing the current view.
  • Export CSV / Import CSV / CSV Template: Move your library in and out of spreadsheet format. Template downloads a blank CSV with all correct column headers.

🗄️ System

  • Backup ZIP: Packs your entire GameManagerConfig folder — database, covers, logos, screenshots, trailers — into a single ZIP archive. This is your complete migration package.
  • Restore ZIP: Unpacks a backup archive. Warning: this overwrites your current library entirely.
  • Add to Application Menu: Registers both CNGM and CREMA in your desktop launcher (GNOME, KDE, XFCE, etc.). Writes .desktop files, extracts icons, and refreshes the launcher database — no terminal required. Re-run it if you move the app folder.
  • Show Welcome & Setup Guide: Reopens the first-run Welcome Screen at any time. Also resets the “don’t show again” flag, so the screen will appear again on the next launch until you opt out again.

🧹 Cleanup

  • Clean Images: Cross-references every image assigned in the database and deletes orphaned files. Safe and surgical — nothing assigned to a current game is touched.
  • Clear All Images: The nuclear option. Wipes all downloaded artwork and screenshots from disk. Your text metadata is preserved, but all visual assets must be re-fetched.
  • Wipe Secure Browser Data: Clears the internal Chromium cache used by the GOG login window and Steam API popups. First thing to try if a store sync is behaving strangely.

10. CREMA — Overview

What Is CREMA?

CREMA is the sweet foam on top of the coffee. It is CNGM’s fullscreen companion app — laidback, easygoing, and focused purely on fun. Designed for couch play with a gamepad, CREMA presents your library in a bold, beautiful TV-mode interface with ambient music integration. Grab a controller and enjoy.

Launching CREMA

Place CREMA.AppImage (any capitalization: .AppImage, .appimage) in the same folder as CNGM.AppImage. A Go Fullscreen With CREMA button appears at the bottom of CNGM’s sidebar. Clicking it minimizes CNGM and launches CREMA as a separate fullscreen process.

CREMA can also be launched independently — from your desktop launcher, terminal, or as an autostart entry. It always reads from and writes to the shared GameManagerConfig/game_library.db in the same folder, so the library is always in sync.

Shared Library

CNGM and CREMA always access the same SQLite database file. Any game you add, scrape, or edit in CNGM is immediately visible in CREMA. Install status updates from Steam are pushed live to both apps simultaneously via a background file watcher — no manual action required.

12. CREMA — Gallery & Gamepage

The Gallery

CREMA’s main view is a fullscreen cover-art grid. Categories are shown at the top — navigate between them with LB/RB (gamepad) or Left/Right arrow keys. The grid reloads instantly for the selected category.

Available Categories in CREMA

  • All Games, Installed — full library or just installed games
  • Steam, GOG, Epic, Amazon — by store
  • Others, Physical, Emulation, Apps — manual categories
  • Want to Play, Favs — your personal lists

The Gamepage

Select a game and press A / Enter to open its Gamepage. You’ll see:

  • Hero Banner — cinematic, takes up ~40% of screen height. The transparent Logo is overlaid at center.
  • Stats Panel — Release Year, HLTB, ProtonDB, Genre, Co-op, Players, Metacritic.
  • Description — a short summary of the game.

  • Trailer Button — appears only if a trailer was downloaded in CNGM. Plays instantly, offline. BGM fades out automatically.

FAV & Want to Play

From the Gamepage menu, you can toggle Favorites and Want to Play. Changes write instantly to the shared database and are reflected in CNGM immediately.

13. CREMA — Scraping & Metadata

CREMA can scrape metadata for individual games without going through CNGM. The scope matches CNGM’s Auto-Fetch: Cover, Hero Art, Logo, HLTB, ProtonDB, Description, Genre, Developer, Publisher, Co-op, Players, Metacritic, Screenshots.

How to Scrape

Navigate to a game in the gallery and press X / Square (or use the scrape option in the Game Menu). CREMA will:

  1. Search Steam for the game by name. If found, proceed with a full Steam + IGDB scrape.
  2. If Steam search finds nothing or the result looks wrong, CREMA automatically offers an IGDB fallback — a search overlay where you can type the correct game name and select from results.

IGDB Fallback

When the Steam search fails, a search overlay appears with results from IGDB. Select the correct entry and CREMA fetches all available metadata. This is especially useful for older games, non-English titles, or games not on Steam at all.

IGDB credentials required: The IGDB fallback path requires a Twitch dev API key configured in CNGM’s Connect menu. Without it, only Steam-based data is available.

14. CREMA — Music & Atmosphere

CREMA has a built-in music player for ambient game-room atmosphere. It runs continuously while you browse your library and integrates seamlessly with the rest of the experience.

BGM Categories

CREMA ships with curated music organized into genre and mood categories. The default on first launch is Ambient. Switch between categories in the Sound Settings overlay (Start button or the on-screen menu icon).

Sound Settings Controls

  • Switch BGM category / style
  • Adjust BGM volume
  • Toggle BGM on or off
  • Adjust sound effects volume

Trailer Fade

When you play a trailer from the Gamepage, CREMA automatically fades out the BGM. When the trailer ends or you close the player, the BGM fades smoothly back in. This ensures a seamless, cinematic experience without jarring audio overlap.
